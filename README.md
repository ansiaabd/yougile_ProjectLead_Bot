# TaskBot — менеджер задач через Telegram

Telegram-бот для управления задачами с ролевой моделью (админ, модератор, пользователь), подтверждением выполнения, вложением файлов/комментариев к выполнению и уведомлениями о просрочках.

## Возможности

- **Ролевая модель** — админ, модератор, пользователь. Модератора назначает админ
- **Регистрация** — `/start`, бот запоминает пользователя для уведомлений
- **Создание задач** — через `/add` (пошагово, с выбором исполнителя из списка) или одной строкой
- **Inline-формат** — `задача title / description / deadline / @user` — дедлайн определяется автоматически в любом месте текста, URL не ломаются
- **Умный парсинг дат** — «11 июня в 14», «завтра в 15», «сегодня в 20», «08.06 18:00»
- **Выбор исполнителя** — inline-кнопки со всеми пользователями, включая себя
- **Комментарий и файл к выполнению** — после отметки «выполнено» бот предлагает добавить 💬 комментарий или 📎 файл, затем отправить на подтверждение
- **Подтверждение выполнения** — исполнитель → создатель задачи или админ подтверждает (✅/❌)
- **Подтверждение с деталями** — админ/создатель видит комментарий и файл исполнителя в уведомлении
- **Напоминания** — каждую минуту проверяет дедлайны, уведомляет администратора и исполнителя
- **Быстрое меню** — `/menu` с inline-кнопками (Мои задачи, Выполнено, Просрочки, Помощь)
- **Natural language** — «выполнено», «сделано», «готово» отмечают активную задачу
- **Inline-кнопки** на каждом сообщении: ✅ Выполнено, 🗑 Удалить, ✅ Подтвердить, ❌ Отклонить

## Команды

### Для всех
| Команда | Описание |
|---------|----------|
| `/start` | Зарегистрироваться |
| `/add` | Создать задачу (пошагово) |
| `/list` | Мои задачи |
| `/done <id>` | Отметить выполненной + добавить комментарий/файл |
| `/delete <id>` | Удалить задачу |
| `/overdue` | Просроченные задачи |
| `/menu` | Быстрое меню |
| `/cancel` | Отменить создание |
| `/help` | Справка |
| «выполнено» / «сделано» / «готово» | Быстрая отметка |

### Админ и модератор
| Команда | Описание |
|---------|----------|
| `/pending` | Задачи на подтверждении |
| `/list all` | Все задачи (только админ) |

### Админ
| Команда | Описание |
|---------|----------|
| `/users` | Список пользователей с ролями |
| `/removeuser <id>` | Удалить пользователя |
| `/promote <id>` | Назначить модератором |
| `/demote <id>` | Снять модератора |

## Роли
| Роль | Видит задачи | Может подтверждать |
|------|-------------|-------------------|
| 👑 **Админ** | Все | Любые |
| 🛡 **Модератор** | Назначенные ему + созданные им | Только свои |
| 👤 **Пользователь** | Только свои | — (только отмечает done) |

## Форматы даты
| Ввод | Результат |
|------|-----------|
| `11 июня в 14` | 2026-06-11 14:00 |
| `завтра в 15` | завтра 15:00 |
| `сегодня в 20` | сегодня 20:00 |
| `08.06 18:30` | 8 июня 18:30 |
| `в 14 часов` | сегодня 14:00 |
| `12:30` | сегодня 12:30 |

## Пример работы
```
Пользователь:  /add → выбирает исполнителя → вводит:
               hecny-cis.com ссылки/подбираем доноров
               https://docs.google.com/...  11 июня в 14
Бот:          ✅ Задача #5 добавлена
               ⏰ 11 июня 2026 в 14:00 | 👤 @user

Пользователь:  /done 5 → 💬 Комментарий | 📎 Файл | 📤 Отправить
Пользователь:  пишет комментарий → 📎 прикрепляет файл → 📤 Отправить
Бот админу:   📬 Исполнитель отметил задачу #5
               💬 Готово, проверил всё
               🖼 Фото прикреплён
               [✅ Подтвердить] [❌ Отклонить]
```

## Установка
```bash
git clone https://github.com/ansiaabd/telegram-task-bot.git
cd telegram-task-bot
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
Создать `.env`:
```
BOT_TOKEN=токен_от_@BotFather
ADMIN_ID=ваш_telegram_id
```
Запуск:
```bash
python main.py
```

## Стек
- Python 3.9+ / python-telegram-bot 22.x
- SQLite (WAL mode)
- APScheduler (JobQueue)
