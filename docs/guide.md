# TaskBot — инструкция пользователя

Telegram-бот для управления задачами. Поддерживает роли: администратор, модератор, пользователь.

---

## 📝 Для всех пользователей

### Начать работу
Напишите `/start` — бот зарегистрирует вас в системе. Теперь администратор или модератор может назначать вам задачи.

### Создать задачу
**Способ 1 — пошагово:**
Напишите `/add` или просто слово «задача»:
1. Выберите исполнителя из списка (или нажмите «Себя»)
2. Введите название
3. Введите описание (или `/skip`)
4. Введите дедлайн → задача создана

**Способ 2 — одной строкой:**
```
задача / Название / Дедлайн / @исполнитель
задача / Название / Описание / Дедлайн / @исполнитель
задача Название / Дедлайн / @исполнитель
```
Дедлайн можно указать в любом месте после названия.

### Мои задачи
`/list` — показывает задачи, назначенные вам.

### Отметить выполнение
- `/done <ID>` — например `/done 3`
- Или напишите «выполнено», «сделано» или «готово» — если у вас одна активная задача, она отправится на подтверждение

### Удалить задачу
`/delete <ID>` — удалить задачу.

### Просроченные задачи
`/overdue` — список просроченных задач.

### Быстрое меню
`/menu` — показывает кнопки с основными действиями.

### Отмена
`/cancel` — отменить создание задачи, если бот застрял в пошаговом режиме.

---

## 🛡 Для модератора

Модератора назначает администратор командой `/promote <ID>`.

Модератор может:
- Создавать задачи любым пользователям (как обычный пользователь)
- **Подтверждать** выполнение задач, которые он создал
- **Отклонять** отметки о выполнении
- Просматривать задачи, которые он создал ИЛИ назначены ему

### Просмотр своих задач
`/list` — показывает задачи, где вы исполнитель ИЛИ создатель.

### Задачи на подтверждении
`/pending` — показывает задачи, ожидающие вашего подтверждения (только те, что вы создали).

### Подтвердить / отклонить
Когда исполнитель отмечает задачу «выполнено», вам приходит уведомление с кнопками:
- ✅ **Подтвердить** — задача получает статус «выполнена»
- ❌ **Отклонить** — задача возвращается в активные

## 👑 Для администратора

Всё, что могут модератор и пользователь, плюс:

| Команда | Описание |
|---------|----------|
| `/list all` | Все активные задачи всех пользователей |
| `/users` | Список всех пользователей с ролями |
| `/removeuser <ID>` | Удалить пользователя |
| `/promote <ID>` | Назначить модератором |
| `/demote <ID>` | Снять модератора |

Администратор может подтверждать и отклонять любые задачи, не только свои.

## 📅 Форматы даты (дедлайн)

| Как писать | Результат |
|------------|-----------|
| `8 июня в 12` | 8 июня 12:00 |
| `завтра в 15` | завтра 15:00 |
| `сегодня в 20` | сегодня 20:00 |
| `08.06 18:30` | 8 июня 18:30 |
| `в 14 часов` | сегодня 14:00 |
| `12:30` | сегодня 12:30 |
| `08.06.2026 16:00` | 8 июня 2026 16:00 |

## 💬 Примеры

**Создать задачу себе:**
```
задача Купить молоко / сегодня в 20 / @myname
```

**Создать задачу коллеге (модератор):**
```
/add → выбрать @ivan → «Отчёт» → /skip → «завтра в 10»
```

**Отметить выполнение:**
```
/done 5
```

или просто:

```
выполнено
```

## ❓ Частые вопросы

**Бот не отвечает на /start?**
Возможно, бот перезапускается. Подождите 10 секунд и попробуйте снова.

**Написал число, а бот создал задачу**
Значит, был активен незавершённый `/add`. Напишите `/cancel` и начните заново.

**Не могу найти задачу в /list**
Проверьте: вы видите только те задачи, где вы исполнитель. Если задача создана вами для другого человека, она видна только вам (модератору/админу).
