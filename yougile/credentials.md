# Доступы для интеграции с Yougile

Заполните значения в файле `.env` в корне проекта.

## 1. COMPANY_ID (ID компании)
Где взять: **Yougile → Настройки компании → API**
Строка вида: `80eed1bd-eda3-4991-ac17-09d28566749d`

## 2. API_KEY (API-ключ)
Где взять: **Yougile → Настройки компании → API → Ключи API → Создать ключ**
Потребуется логин/пароль от админ-аккаунта Yougile + Company ID.
Строка вида: `-VRKgqzejQhROjMTmOk3h20Rnj6XKkwBFx5GkK1fA34TvLFmiVdmEPt3-ygeWfiy`

## 3. WEBHOOK_SECRET (для обратной синхронизации)
Для синхронизации Yougile → Telegram. Пока не нужно, заполним позже.

## Примечание
PROJECT_ID и COLUMN_* теперь не нужны в `.env` —  
при создании задачи бот сам получает список проектов из Yougile,  
ты выбираешь нужный, и задача создаётся в первой доске этого проекта.  
Колонки определяются автоматически (первая колонка → активные, «Готово» → выполненные).

## Куда вставлять

Эти строки уже в файле `yougile/.env` (рядом с BOT_TOKEN в `.env` тоже можно продублировать):

```
YOUGILE_COMPANY_ID=ваш_id_компании
YOUGILE_API_KEY=ваш_api_ключ
```
